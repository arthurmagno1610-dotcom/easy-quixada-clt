# EasyQuixada Cliente

Protótipo mobile-first inspirado no frame 14 do Figma, com listagem de estabelecimentos de Quixadá e tela de detalhes para cada estabelecimento.

## Como abrir

Abra `index.html` no navegador. O projeto não precisa de build nem instalação de dependências.

## Fluxo implementado

- Home mobile com topo, categorias, busca e lista de estabelecimentos.
- Clique em um estabelecimento para abrir a tela de detalhes.
- Detalhes com imagem, categoria, endereço, horário, telefone, Instagram e ação de ligação.
- Navegação por hash (`#/` e `#/estabelecimento/:id`).
